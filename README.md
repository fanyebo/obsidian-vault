# Obsidian Vault — 个人知识库

## 目录结构

- `00-Inbox/` — 临时笔记，待整理
- `10-Notes/` — 技术笔记、项目记录
- `20-Projects/` — 项目相关文档
- `30-MOCs/` — 内容地图（Map of Content）
- `99-Archive/` — 归档

## 工作流

1. 新想法 → `00-Inbox/`
2. 整理后 → `10-Notes/` 或 `20-Projects/`
3. 关联到 → `30-MOCs/`
4. 不再活跃 → `99-Archive/`

## 同步

- 本地编辑，Obsidian Git 插件自动 commit + push
- 新电脑：`git clone` → Obsidian 打开 vault
