# 个人知识库架构

> 2026-06-29 搭建完成

## 方案：Obsidian + Git + GitHub

| 组件 | 路径/地址 |
|------|----------|
| Obsidian | `D:\software\Obsidian\Obsidian.exe` |
| Vault | `C:\Users\fanye\Documents\Obsidian Vault` |
| Git 远程 | `git@github.com:fanyebo/obsidian-vault.git`（私有） |
| 自动同步 | Obsidian Git 插件，每 10 分钟 commit + push |
| 代理 | SSH 通过 Clash `127.0.0.1:7897` 走 SOCKS5 |

## 新电脑恢复

```bash
git clone git@github.com:fanyebo/obsidian-vault.git "C:\Users\fanye\Documents\Obsidian Vault"
# 打开 Obsidian → 自动识别已有 Vault
```

## 与 Hindsight 的分工

| Hindsight | Obsidian |
|-----------|----------|
| 自动从对话提取事实/决策/偏好 | 手动撰写长文档、SOP、笔记 |
| 语义检索 + 推理合成 | 全文搜索 + 目录组织 |
| 云端，换电脑 API Key 恢复 | 云端（GitHub），clone 恢复 |

## 放弃的方案

- **飞书做同步层**：格式转换损耗（md ↔ 富文本），API 复杂度高，不如 Git 直接
- **仅本地存储**：换电脑丢失，需云同步层
